一. Motivation

However, most existing truth discovery methods are designed for structured data, and cannot meet the strong need to extract trustworthy information from raw text data as text data has its unique characteristics.

most existing truth discovery methods are designed for structured data, and are difficult to be directly applied to text data, which are unstructured and noisy. This significantly narrows the application domain of these truth discovery methods, as a large ratio of the multi-sourced data are text. Actually, there are several unique characteristics of natural language that hinder the existing truth discovery methods from being successfully applied to text data.

大多数现存的真值发现方法是针对结构化数据的, 由于文本数据其独特的特性, 其并不能很好的满足从原始文本数据中提取可信信息的强烈需求(其阻碍了真值发现方法成功地应用在文本数据).

 

文本数据的两个特征

 

文献中例举了一个例子: 流感的特征都有些什么?

正确答案应该包含很多因素: fever, chills, cough, nasal symptom, ache, and fatigue(发烧、寒战、咳嗽、鼻部症状、疼痛和疲劳, 流鼻涕)等

但对于某个用户所提交的答案可能仅仅包含两个因素: cough and chills

 

First, the answer to a factoid question may be multifactorial, and it is usually hard for a given text answer to cover all the factors. For the question ‘What are the symptoms of flu?’, the correct answer should contain the following factors: fever, chills, cough, nasal symptom, ache, and fatigue. Even if the answer provided by a user covers two factors, such as cough and chills, the existing truth discovery methods may determine this answer to be totally wrong and assign a low reliability degree to this user. This is because these methods treat the whole answer as an integrated unit. However, if we take the fine-grained(细粒度) answer factors into consideration, the answer provided by this user is partially correct, which implies that we should give some credits to the user by increasing his/her reliability degree. Thus, how to identify partially correct answers and model factors of text answers is critical for the task of truth discovery on text data.

现存的真值发现方法对于上述用户的答案可能会完全判定为否定从而给予其回答用户一个低的可靠度, 因为这些方法将答案作为一个完整的单元. 如果我们考虑这些答案的细粒度, 即考虑该用户提供的答案是部分正确的, 这意味着我们应该增加一些该用户的可靠度.

鉴于上面的特性, 对于基于文本数据的真值发现任务重要的是如何鉴别部分正确的答案同时建模文本答案的因素们

The second characteristic of text data is the diversity of word usages. Answers provided by online users may convey a very similar meaning with different keywords. For example, users may use words such as tired or exhausted to describe the symptom of fatigue. However, existing truth discovery approaches may treat them as totally different answers. Thus, it is of great importance to model the diversity among answers in the text data when inferring trustworthy information.

当然用户也可能回答类似的意思但用了不同的单词来描述, 比如说使用 tired 和 exhausted 来描述流感症状中的疲惫现象, 对于现存的真值发现方法, 对于上面两个回答可能定义为不同的答案.

所以, 在文本数据中推断可信信息时, 建模答案们的多样性是非常重要的.

 

 

综上所述, 文本数据的两个特征为:

  1. 对于一个问题的答案, 其可能是多因素的, 同时对于一个给定的文本答案, 其很难覆盖所有因素.
  2. 文本信息中单词用法的多样性

The proposed method works in an unsupervised manner, and thus can be applied to various application scenarios that involve text data. Experiments on three real-world datasets show that the proposed TextTruth model can accurately select trustworthy answers, even when these answers are formed by multiple factors.

Second, by modeling answer keywords in the form of vector representations, we can make the factors within the answers computable such that the ubiquitous usage diversity issue on text data is addressed.

  1. 能够很好的解决在文本答案中部分正确的现象

五. Methods

we propose a model named “TextTruth”, which takes the keywords in each answer as inputs and outputs a ranking for the answer candidates based on their trustworthiness. Specifically, we first transform the keywords in text answers into pre-trained computable vector representations. Due to the fact that an answer may contain multiple factors, the “answer-level” or coarse-grained(粗粒度) representations may not be able to capture the partially correct answers. Thus, we need to convert the whole answer into fine-grained(细粒度) factors. Then, we model the diversity of answers by clustering the keywords with similar semantic meanings. By doing so, we can estimate the trustworthiness of each answer factor instead of the whole answer and infer the correctness of each factor in the answer.

TextTruth 方法将每一个答案中的关键词作为输入, 并基于候选答案的可信度输出一个排名.

具体来说, 首先将文本答案中的关键词转换为预先训练可计算的向量表示. 由于一个答案可能包含多个因素, "答案水平" 或者 粗粒度表示可能不能够捕获部分正确答案, 所以我们需要将整个答案转变为细粒度因素.

之后, 我们通过聚集带有相似语义的关键词来建模答案的多样性.

通过上面的做法, 我们评估的是每个答案中的因素们的可信度而不是整个答案,同时来推断答案中每个因素的正确性.

Overview

In this paper, for each question, we first extract the keywords in each answer and learn their vector representations. Then we cluster these word/phrase-level keywords into semantic clusters(i.e., factors). These factors represent all the possible key points in the answers to a question and can be used to determine the trustworthiness of an answer. For the keywords within each cluster, as they share very similar semantic meanings, their trustworthiness should be almost the same. In addition, users may have different reliabilities, which can be reflected in the answers they provided.

对于每个问题, 我们首先提取每个答案的关键词并学习他们的向量表示, 然后聚集这些单词/短语关键词到语义聚类(即因素)中. 这些答案因素代表了答案中所有可能的关键点并且被用来决定一个答案的可信度.

对于每个聚类的关键词们, 因为他们有着相同的语义,所以他们的可信度应该是几乎一样的. 此外, 用户可能有不同的可靠度, 可靠度可以有他们所提供的答案中来反映.

 

Based on the above ideas, we propose a two-step method to estimate the trustworthiness of each answer. In the first step, we specify a probabilistic model to model the generation of keywords with user reliabilities taken into consideration in Section 4.2. The generative model, which consists of three major components, jointly learns the answer factors and their truth label. The generative model first generates a mixture of answer factors and their semantic parameters. After that, the model generates two-fold user reliability variables, which model the comprehensiveness and accuracy of answer factors provided by a specific user.These two variables capture a whole spectrum of the user reliability. Finally, the model selects an answer factor based on the semantics, the trustworthiness of the answer factor as well as the reliability of the user that provides the answer, and generate the keyword embedding vector via a von Mises-Fisher (vMF) distribution.

文献提出了一个两步骤方法来评估每个答案的可信度.

在第一步中, 文献指定了一个概率模型考虑用户可靠性来对关键词的产生来建模, 该模型由三部分组成.

第二步, 模型生成双重用户可靠度变量, 这个变量建模由具体用户提供的答案因素的全面性和准确性.全面性和准确性这两个变量捕获用户可靠性的整个光谱. 最终, 模型基于语义挑选一个答案因素, 答案因素的可信度和用户的可靠度 以及通过vMF分布生成关键字嵌入向量。

Results

七. Conclusions

As an emerging topic, truth discovery has shown its effectiveness in a wide range of applications with structured data. However, existing methods all suffer on unstructured text data, due to the semantic ambiguity of natural languages and the complexity of text answers. To tackle these challenges, in this paper, we propose a probabilistic model named TextTruth that takes vector representations of key factors extracted from answers as inputs and outputs the ranking of answers based on the trustworthiness of key factors within each answer. Specifically, the model jointly learns the clustering label and truth label for each answer factor cluster through modeling the generative process of answer factors’ embedding representations. Experimental results on three real-world datasets prove the effectiveness of the proposed TextTruth model. Furthermore, case studies illustrate that the learned labels are interpretable.

 

为了解决自然语言的语义问题和文本答案的复杂性, 提出了一个叫做 TextTruth的概论模型. 相对应的实验表明这个模型的优越性.
